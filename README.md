# 謝遠青 - Personal Autobiography Website

A modern, responsive website showcasing an autobiography with smooth animations and intuitive navigation.

## 📋 Features

- **Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices
- **Smooth Navigation** - Sticky navbar with smooth scrolling to sections
- **Animated Elements** - Fade-in animations as you scroll through the page
- **Clean Layout** - Professional design with clear sections for different content
- **Multiple Sections**:
  - Hero introduction
  - About & personal information
  - Educational background
  - Hobbies and interests
  - Goals and aspirations
  - Contact section

## 📁 Files

- `index.html` - Main HTML structure
- `styles.css` - Complete styling and responsive layout
- `script.js` - JavaScript for interactivity and animations
- `README.md` - This file

## 🚀 Getting Started

### Option 1: Local Development
1. Clone or download these files
2. Open `index.html` in your web browser
3. No build tools or server needed!

### Option 2: GitHub Pages
1. Go to repository Settings → Pages
2. Select the branch and folder where files are located
3. Your site will be live at `https://wellflix.github.io/Wellautobiography`

## 🎨 Customization

### Change Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #2c3e50;      /* Main color */
    --secondary-color: #3498db;    /* Accent color */
    --accent-color: #e74c3c;       /* Highlight color */
    --light-bg: #ecf0f1;           /* Light background */
}
```

### Update Personal Information
1. Open `index.html`
2. Find the section you want to edit
3. Update the text content
4. Save and refresh the browser

## 📱 Responsive Breakpoints

- **Mobile (480px)**: Optimized for small screens with compact navigation
